# charmcity-electron

A simple text editor built with [Electron](https://github.com/atom/electron) and [Ace](https://github.com/ajaxorg/ace)
shown at [charmCityJS](http://charmcityjs.com).

## Develop

    npm install
    npm install --g electron-prebuilt gulp
    gulp build-dev
    npm start

## Package as an app

    gulp package
